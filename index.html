Syntax und Unterschiede zu Java

  • Keine Semikolons
  • Keine "unnötigen" (geschweiften) Klammern
    class EineKlasse // korrekt!
  • Typen-Inferenz
  • Public by default
  • Keine Vererbung by default
    class KannNichtErben
class UnterKlasse : KannNichtErben // fehler!
								
    open class KannErben
class UnterKlasse : KannErben // erlaubt
